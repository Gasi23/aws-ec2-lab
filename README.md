# ☁️ Desafio DIO - Gerenciamento de Instâncias EC2 na AWS

## 📚 Sobre o Projeto

Este repositório foi desenvolvido como parte do desafio prático da DIO com foco em serviços da AWS, especialmente gerenciamento de instâncias EC2.

O objetivo do laboratório foi aplicar conceitos fundamentais de computação em nuvem utilizando serviços da Amazon Web Services, além de praticar documentação técnica utilizando GitHub e Markdown.

---

## 🎯 Objetivos de Aprendizagem

* Aplicar conceitos de computação em nuvem em ambiente prático
* Entender o funcionamento de instâncias EC2
* Utilizar armazenamento EBS e S3
* Configurar regras de acesso e segurança
* Documentar processos técnicos utilizando Markdown
* Utilizar Git e GitHub para versionamento

---

## ☁️ Serviços AWS Estudados

### 🖥️ Amazon EC2

* Criação de instâncias
* Tipos de instâncias
* Elastic IP
* Security Groups
* Modelo de responsabilidade compartilhada

### 💾 Amazon EBS

* Armazenamento persistente
* Volumes em bloco
* Uso em bancos de dados e aplicações web

### 🗂️ Amazon S3

* Buckets
* Versionamento
* Classes de armazenamento
* Lifecycle
* Integração com AWS Lambda

### 💰 AWS Pricing

* AWS Pricing Calculator
* Modelo de cobrança sob demanda
* Instâncias Spot
